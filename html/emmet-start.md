# Emmet (Zen Coding)
- HTML, CSS 코드 작성 생산성 향상 도구  
태그 작성 속도 크게 향상

## Emmet Syntax 문법
- `div.container>a>{hello}`  + tab키
```
<div class='container'>
    <a href=''>hello</a>
</div>
```  
위 두 코드는 같다
